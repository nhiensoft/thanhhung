1) TỔNG QUAN EM HIỂU VỀ WEBSITE
============================================================

Đây là website giới thiệu theo phong cách storytelling về:
- Việt Nam: lịch sử, di sản, làng nghề, ẩm thực.
- Chuyển mạch sang Đại học Mở Hà Nội (HOU): triết lý “5 Mở”, ngành đào tạo, form đăng ký tư vấn.

Website có 2 luồng trải nghiệm:
- Desktop/tablet lớn: dùng bộ component chuẩn ở src/components/* + src/components/sections/*.
- Mobile: có nhiều section mobile riêng ở src/components/mobile/* (header, hero, history, HOU).

Điểm nổi bật kỹ thuật hiện tại:
- React + Vite + TypeScript + Tailwind + shadcn/ui.
- Landing page dạng one-page, dùng anchor menu để cuộn tới section.
- Có preloader, chatbot nổi (gọi Gemini trực tiếp từ client), nút đăng ký tư vấn nổi.
- Dữ liệu nội dung chính nằm trong code TS (không CMS), tập trung ở:
  - src/data/vietnamHeritage.ts
  - src/data/houPrograms.ts
  - Một số section hard-code trực tiếp trong component.
